Polyline Art Generator

Generate your own polyline art!

Fill in a couple of numbers, give it a name, and hit generate!

Instructions

Get Started

Hello and Welcome! The generator will generate random points in a 3D space based on the number of vertices you give it, and draw the given number of triangles, and lastly a name for your art piece. The newly generated drawing would be add to the table below. Click on "View" and the drawing will be drawn in the box to the right.

Neat Features

There are some animations you can toggle on and off! You can make the polyline drawing translate in the x, y, z, directions and also make the drawing rotate! To turn off the animation, just hit the same key again:

Keys Function
! Object moves to the right
@ Object moves to the left
# Object moves up
$ Object moves down
% Object moves closer to you
^ Object moves away from you
& Object rotates
